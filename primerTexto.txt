TFG

Mi proyecto:

Mi idea es un gestor de gastos económicos con base de datos y  login, cada usuario podrá meter sus gastos personales (en el futuro quizás explore opciones como la api de Google pay para hacerlo de manera automática) y clasificarlos con etiquetas. 

Los gastos se podrán editar y confirmar después de hechos.

Al final del mes la aplicación guardará los registros de los gastos de cada mes en la base de datos/api y se podrá entrar a cada mes para ver un gráfico de gastos según sus etiquetas "ocio", "primera necesidad", etc.

En cuanto a las etiquetas, habrá algunas creadas de manera estándar, aunque se podrán crear de manera personalizada.

Tecnologías a usar: HTML, CSS, JavaScript y PHP, además de un gestor de bases de datos (MySql)
